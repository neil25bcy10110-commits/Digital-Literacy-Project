📌 Project Overview
This project focuses on developing digital literacy skills and promoting responsible online behaviour among students. It includes practical tasks such as creating awareness resources, building a professional digital portfolio, exploring useful platforms, practising email etiquette, and understanding cybercrime prevention.
The goal of this project is to help students use digital tools effectively, communicate professionally, and stay safe online in both academic and professional environments.
